HTML - (HyperText Markup Language) язык гиппер текстовой разметки.

<html></html> - основной тег разметки, где записывается абсолютно все, что касается разметки;
<head></head> - часть страницы где подключаются скрипты, описывается мета информация и таблицы стилей;
<body></body> - тег разметки ТЕЛА страницы;

ВАЖНО: Каждый тег в конце должен замыкаться, с помощю "/".

<title></title> - главный заголовок сайта;

<!DOCtype html> - начало кода/разметки;

<meta charset="uft-8" /> - мета тег с функцыей charset, - набор символов кодировки, utf-8 один из самых популярных, там есть латиница и кирилица и некоторые дополнительные символы;
uft-8 - русский язык;

<hХ></hХ> - тег заголовка, Х от 1 до 6, в соответствии от размера заголовка;
<p></p> - абзац, усли не применять этот тег - то весь текст будет сплошным (в одну строку);
	атрибут: <p align="left(по умолчанию), right, center, ..." - расположение абаца;
<!-- Koments... --> - одн из спобов представления коментариев;
<b></b> vs <strong></strong> - жирный текст;
<i></i> vs <em></em> - курсив;
<strike></strike> - перечеркивание текста;
<u></u> - подчеркнутый текст;
<abbr></abbr> - абривиатура:
	пример: <p><abbr title="Hyper Text Mark-up Language">HTML</abbr></p>
<address></address> - тег адрес, тот же курсив, но поисковый робот будет лучше идентифицыровать страницу;
<small></small> - текст в этом теге меньше основого текста;
<big></big> - увелечине текста;
<pre></pre> - тег котрый сохраняет все пробелы, отступы и переходы на другую строку, сам по себе означает, что текст уже отформатирован, в него код удобно писать;
<blockquote></blockquote> - вывод обёмной цитаты;
	атрибут: cite - обясняет поисковому роботу что текст не украден, а цитируется;
		пример: <blockquote cite="https://vk.com/citationsgreat">
<q></q> - тег цытаты с кавычками, так же имеет атрибут cite;
<hr /> - тег который проводит горизонтальную линию;
<br /> - образует пустую строку;
<nobr></nobr> - для отображения текста в одну строку;

Теги для создания стисков и єлементов списков
<ol></ol> - пронумерованый список;
	<li></li> - (list item) элемент списка;
	атрибут: type - изменяет тип нумерации, пример <ol type="a">, по умолчанию задано цифры;
		 start - с какого числа будет начинатся нумерацыя, пример: start="3" ==> 3, 4, 5, 6...;
		 reversed - вывод нумерации списка в обратном порядке, reversed="reversed" ==> 3, 2, 1;
<ul></ul> - непронумерованый список;
	<li></li> -элемент списка;
	атрибут: type="square,  disc(по умолчанию), circle" - изменяет вид списка;
<dir></dir> - список который будет иметь больший отступ чем ul, устаревший вариант;

Атрибуты
<p align="left(по умолчанию), right, center, ..." - рамположение абаца;
title - при наведении будет высвечиваться подсказка, подходит к любому тегу;

Сылки
<a href="ссылка"></a> - тег для назнаения ссылки;
	атрибуты: target="_blank (отрыте в новом окне), _self(по умолчанию, открытие в этом же окне)" - атрибут для открытия ссылки в новом окне;
		  title - при наведении будет высвечиваться подсказка;
		  href="#..." - перезагрузки страницы на будет, для переходов по странице;
			пример:<a href="#example">Переход</a>...... <a name="examle"></a>;
Вставка изображений
<img src="ссылка" /> - вставка изображений;
	атрибуты: width="100px"  - ширина картинки;
		  height="100%" - высота картинки;
		  alt="text..." - показывает текст в случае если картинка не отображаеться;
		  border="x px" - размер рамки;
		  align - расположение;

Таблицы
<table></table> - основное тело таблицы;
	<caption></caption> - название таблицы;
	<tr></tr> - название столбцов;
		 <th></th> - название столбца;
		 <td></td> - задает значения рядка столбца;
	атрибуты: border - толщина границы, по умолчанию 0;
		  cellpadding - отступ;
		  cellspacing - пробелы;

<td></td> атрибуты: colspan - показывает сколько столбцов будет занимать этот столбец, по умолчанию 1;
		    bgcolor - фон, по умолчанию белый;
		    align - размещение текста, по умолчанию слева;
		    valign="top(сверху), middle(по центру), botton(внизу)" - размещение текста по вертикали;
		    width - ширина;

Подключение файлов
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8" /> - тег с атрибутами который указывает на кодировку UTF-8, более професиональный способ кодировки текста;
<meta name="description" content="описание" /> - тег описание сайта для поискового робота и поисковый систем, рекомендуется от 100 до 200 символов;
<meta name="keywords" content="ключевые слова" /> - ключевые слова сайта;

<link href="ссылка" rel="stylesheet" type="text/css" /> - так подключения стлей;
<script src="пусть к скрипту"></script> - подключение стриптов;

Теги div and span
<div></div> - тег создания блока;
<span></span> - тег создания блока;

Поля ввода
<form></form> - тег отправки(создание) форм;
	атрибути: name - атрибут мени, с помощю его можно ссылаться на эту форму;
		  action - гдe будут обрабатываться данные формы;
		  method - метод отправки, понадобиться когда буду использовать php;
	<input /> - тег ввода, TextBox(C#)
		атрибуты: type="reset(очистить все поля), text, url, button(кнопка для JS, без перезагрузки страницы), submit(кнопка для php, с перезагрузкой страницы), tel(телефон), range(позунок), radio(радіокнопка), password, number, file(выбор файла, например изображения), email, color(плагин выбора цвета), checkbox" - атрибут типа воодимого текста;
			  value="значение";
			  placeholder="..." - текст на заднем фоне, как пример вводимого текста или подсказка;
			  maxlength - максимальная длина вводимого текса;
			  id="name";
			  title...;
	<label for="атрибут названия"></laber> - тег надпись;
		пример: <label for="name">Пример текста</lable>
			<label for="date">День рождения: </label><input type="date, text, week(день недели)..." name="date" id="date"/>

Тег textarea
<textarea></textarea> - большое текстовое поле;

Тег button

Селектор выбора (select and option)
<select></select> - делает выпадающий список
	<option></option> - элемент списка;
		атрибуты: selected="selected" - назначение элемента первым, иначе первым будет тот что первый в списке;
			  disabled="disabled" - уберает возможность выбору этого варианта;
			  value - значение которое передается при отправке даных;
	атрибуты: multiple="multiple" - возможность выбору нескольких вариантов, по умолчанию стоит один выбор;
		  size - количество максимального выюору вариантов;
